# Bolinho de Arroz
O **bolinho de arroz** é uma receita que transforma sobras de arroz em pequenas delícias crocantes por fora e macias por dentro!


Os ingredientes são simples e incluem **arroz cozido, ovos, leite, farinha, amido de milho, fermento, queijo e temperos**. Essa é a combinação certeira para ser uma opção versátil para um lanche rápido ou um acompanhamento especial. A simplicidade no preparo e o resultado maravilhoso fazem desta receita uma opção adorada por crianças e adultos. Cada mordida revela acrocância da casquinha dourada combinada com um interior suave, macio e bem temperado que transforma o bolinho de arroz simples em uma verdadeira iguaria. Siga o passo a passo da receita e desfrute do prazer de degustar um apetitoso bolinho de arroz frito. Aproveitar sobras nunca foi tão saboroso!

--- 

## Ingredientes ( para o bolinho )
- **Arroz cozido** : 2 xícaras (chá)
- **Leite** : 1/2 xícara (chá)
- **Fermento em pó** : 1 colher (sopa)
- **Farinha de trigo** : 1/2 xícara (chá)
- **Queijo ralado** : 1/2 xícara (chá)
- **Cheiro Verde** : 2 colheres (sopa)
- **Amido de milho** : 1/2 xícara (chá)
- **Ovos** : 3
